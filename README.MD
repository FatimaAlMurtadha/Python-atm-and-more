# Student Performance Analysis

## Project

This folder contains the KK1 analysis for a student performance dataset.
The analysis is performed in `FinalScoreDS.ipynb` and focuses on how study time, anxiety, stress, and study method relate to final score.
The dataset kan be downloaded directly as csv file here: `` https://www.kaggle.com/datasets/sarveshchhetri/student-lifestyle-vs-academic-performance-dataset/data ``

## Dataset

- File: `student_performance_finalscore.csv`
- Each row represents one student.
- Includes academic variables, psychological measures, lifestyle factors, and a final score.

The data appears to be a synthetic or cohort dataset from a student study, so conclusions are primarily valid for this sample.

## Running the Notebook

1. Activate your Python environment.
2. Install the dependencies:
   ```bash
   pip install pandas matplotlib seaborn numpy
   ```
3. Open `FinalScoreDS.ipynb` in Jupyter or VS Code and run the cells in order or unn them all by clicking the button Run All.

## Contents

- `FinalScoreDS.ipynb` — notebook with loading, inspection, cleaning, visualization, and conclusions.
- `student_performance_finalscore.csv` — dataset used in the analysis.

## Purpose

The notebook is intended to demonstrate skills in:

- pandas for data handling
- numpy for numerical calculations
- matplotlib/seaborn for visualization
- data cleaning and question-driven analysis
- narrative documentation in a Jupyter notebook
